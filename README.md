# TaskForge API

TaskForge is a small FastAPI project management API built as Project 2 in a
20-project backend learning path.

The goal of this project is to move beyond basic CRUD by modelling users,
projects, tasks, permissions, task status transitions, dependency injection,
and audit logs.

The app uses a layered structure where routers handle HTTP, services enforce
business rules, repositories handle persistence, and FastAPI dependencies wire
the application together.w# taskforge
